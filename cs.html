Scratch

This Scratch course teaches kids the fundamentals of Scratch and coding in a block-based language. Ideal for anyone wanting to begin coding or polish their skills, this course is filled with interactive group work and team problem-solving. This course will give kids a complete overview of the website, the blocks, how they work, and use projects and hands-on virtual learning in order for kids to be able to make their own games step-by-step. Key projects in this course include a guess my number game (with sensing), a maze game, choose your own adventure game. Most of the projects will be done in a Scratch Studio which gives kids the ability to view each other's projects. This course is recommended for students in grades 2-5 and is suitable for all skill levels.
  • Module 1: Scratch environment (website overview)
  • Module 2: Motion (coordinate plane overview)
  • Module 3: Looks and Sounds Complete Overview
  • Module 4: Loops
  • Module 5: Conditionals with Sensing
  • Module 6: Conditionals with Operators
  • Module 7: String Manipulation from User Input
  • Module 8 : Variables
  • Module 9: Clones and Broadcasting
  • Module 10: Blocks

Python

This introductory Python course covers the fundamentals of Python, and serves as an introduction to text-based languages. Projects in this course include hangman, tic-tac-toe, rock-paper-scissors, and a shopping game. With smaller classes and guidance from our amazing instructors, students will receive personalized guidance, and an opportunity to connect with others. This course is recommended for highly motivated 6-8th graders, or those who have taken Scratch during last semester.
  • Module 1: Introduction to Python
  • Module 2: Variables and Data Types
  • Module 3: If-Else Statements
  • Module 4: String Methods
  • Module 5: Functions
  • Module 6: While Loops
  • Module 7: For Loops
  • Module 8: Lists
  • Module 9: 2D Arrays and Nested Iteration
  • Module 10: Dictionaries
  • Module 11: Introduction to Python Libraries

Java

Our introductory Java course provides students with a strong foundation in one of the most common programming languages used today. It approaches Java from a functional programming perspective, giving students a true understanding of how the internal structures of Java function. Through projects and games like Mastermind as well as thorough coverage of foundational computer science topics such as data structures and algorithms, this course will set students well on their way to becoming skilled Java developers. It is only recommended for students in grades 6 - 8 with previous text-based programming experience and a strong foundation in math through operations with decimals, fractions, and negative numbers.
  • Module 1: Data Definitions
  • Module 2: Classes
  • Module 3: Interfaces
  • Module 4: Methods
  • Module 5: Lists
  • Module 6: Recursion
  • Module 7: Abstract Classes
  • Module 8: Array Lists
  • Module 9: Loops
  • Module 10: Searching
  • Module 11: Sorting
  • Module 12: Arrays
